## Overview

Affinity Photo Professional Suite is a complete professional imaging platform that covers RAW processing, photo editing, retouching, and compositing in a single application with the depth of tools that demanding creative work requires.

Studio environment with fully customizable panels, tool positions, and keyboard shortcuts allows the interface to be organized exactly as each user's workflow demands, removing the friction of adapting to a fixed layout that does not match how you work.

HDR merge, focus stacking, and panorama stitching tools handle common multi-image techniques with dedicated workflows that produce high-quality results from properly captured source material.

Batch processing applies the same editing operations to any number of files simultaneously, making it practical to apply consistent corrections to large sets of images without editing each file individually.

Affinity Photo Professional Suite is the professional imaging choice for photographers and retouchers who want the full capability of professional image editing software with the ownership model and performance optimization that the best tools on the platform provide.

## Performance & Integration

Affinity Photo is engineered around macOS technologies that handle performance-critical work efficiently. Tasks run in the background without slowing down the rest of your system — you can keep working while Affinity Photo processes operations quietly behind the scenes.

The app integrates deeply with the macOS ecosystem out of the box:

- Launch workflows directly through **Spotlight**
- Preview files instantly with **Quick Look**
- Trigger actions from any app via the **Services menu**
- Fully compatible with **Shortcuts** for automated workflows
